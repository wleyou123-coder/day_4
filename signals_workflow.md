# Работа с сигналами в Linux

## 1. Перечень сигналов
- SIGTERM (15) — корректное завершение
- SIGKILL (9) — принудительное завершение
- SIGINT (2) — прерывание через Ctrl+C
- SIGHUP (1) — перечитать конфиг / мягкая перезагрузка

## 2. Отправка сигналов
kill <PID>
kill -9 <PID>
kill -HUP <PID>
pkill nginx
killall nginx

## 3. Перезапуск сервисов
- Мягкая перезагрузка: SIGHUP
- Полное завершение: SIGTERM
- Принудительное завершение: SIGKILL
